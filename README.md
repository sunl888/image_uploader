# Image Uploader

## Installation

Image Uploader is available via Composer:

```bash
$ composer require wqer1019/image_uploader
```

## Testing

Glide has a [PHPUnit](https://phpunit.de/) test suite. To run the tests, run the following command from the project folder:

```bash
$ phpunit
```
